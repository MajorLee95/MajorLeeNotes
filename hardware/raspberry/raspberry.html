Processeurs¶

  • RPi3 model B BCM2837 à 1.2MHz cortex A53

  • RPI3 model B+ BCM2837 à 1.4MHz

  • Rpi4 : BCM2711 4 coeurs CORTEX A72 version 2Go, 4Go, 8Go

Tous les modèles ci-dessus ont le même facteur de forme. Seuls le RPi 1 A+ et le RPi3 A+ ont un facteur de forme différent

Description complète sur Wikipédia Raspberry pi 2

A titre de comparaison la carte Friendly arm NEO-PC T4 RK3399 2 A72 4 A53 : ARMV8-A (un peu plus de 110€ sur aliexpress au 7/4/2021)

I2C : état des lieux (13/10/2021)¶

i2c-tools:

pip3 install smbus2

en C (safeAir):

pigpio.h
