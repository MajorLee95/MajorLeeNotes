ABI¶

AHB¶

Advanced High-performance Bus 2

AST¶

Abstract Syntax Trees

BPMN¶

Business Process Model and Notation 3

BSP¶

Board Support Package (voir Wikipédia)

CDN¶

Content Delivary Network, réseau de distribution de contenu. Exemple pour bootstrap

CI¶

Continuous Integration (test de non reg automatique): les plus connus CircleCI, Jenkins, CruisControl

CMS¶

Content Managment System : Wordpress, Joomla et Drupal

CRM¶

Customer Relationship Management ou gestion de la relation client

CRUD¶

Creat Read Update Delete : pour les requête API

DLP¶

Digital Light Processing

EFI¶

Extensible Firmware interface successeur du BIOS

FDM¶

Fused deposition modeling - imprimante 3D à filament

FFF¶

Fused filament fabrication

GSM¶

Global System for Mobile Communications (GSM)

HAL¶

Hardware Abstraction Layer

LTE¶

Long Term Evolution (la 4G)

MERN¶

MongoDB, Express js, React, Node.js

Motor Industry Software Reliability Association - standard de dev C et C++

MMU¶

Memory Managment Unit

MOOT¶

Massive Open Online Textbook

SASS¶

SLA¶

SLAM¶

TSD¶

UEFI¶

Unified Extensible Firmware interface successeur de l’EFI
